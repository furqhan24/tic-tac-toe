# Tic-Tac-Toe

A responsive web-based Tic-Tac-Toe game developed using HTML, CSS, and JavaScript. The application provides an interactive two-player gaming experience with real-time move validation, win detection, draw handling, and game reset functionality.

## Features

* Interactive two-player gameplay
* Real-time win and draw detection
* Dynamic game state updates
* Reset and restart functionality
* Responsive user interface
* Clean and intuitive design

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## How It Works

1. Players take turns marking cells on a 3×3 grid.
2. The application validates each move and updates the game state.
3. Winning combinations are checked after every move.
4. The game displays the winner or identifies a draw when applicable.
5. Users can restart the game at any time using the reset functionality.
